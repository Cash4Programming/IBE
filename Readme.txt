Before you compile the project.

Open the Project in VS 2017 (this is what it's created in)

Right click the 'Instructor Briefcase Extractor2' project

and select Properties

On the menu on the left click settings and update the
	SMTPUser
	SMTPPassword

If you use a gmail account you will not need to modify the SMTPServer or SMTPPort.

Now click Publish and change to a location that you have permission to deploy the project to.
